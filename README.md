# PlayerWallet

## Pre-requisites
Java 8+

## How-to Build

### Initialize PlayerWallet
```mvn clean install -Dspring.profiles.active=dev```


## How-to Test
```mvn test -Dspring.profiles.active=dev```

## How-to Run

### 1. Initialize PlayerWallet
```mvn spring-boot:run -Dspring.profiles.active=dev```
